# Dino Reformas

**Tipo:** Proyecto personal  
**Sector:** Gestión de gremios · Reformas en Cantabria  
**Fecha de creación:** 2026-04-23  
**Estado:** en desarrollo

## Concepto
Empresa coordinadora de gremios de construcción. El cliente trata solo con Dino Reformas y nosotros gestionamos a todos los profesionales (albañiles, fontaneros, electricistas, pintores...). Modelo de negocio: margen sobre gremios + material a precio de coste vía BdB Suministros Monte.

## Servicios
- Reforma integral de vivienda (core)
- Reforma integral de cocina
- Reforma integral de baño
- Reformas parciales
- Reparaciones y mantenimiento

## Zona de trabajo
Cantabria

## Stack
- HTML5 + CSS3 + JS vanilla
- Bootstrap 5.3 (CDN)
- Google Fonts: Syne + Inter
- Font Awesome 6 (CDN)
